Projeto RAFABANK

Banco Digital

Resumo

Um projeto 2 em 1 , simulando uma página de apresentação de um banco e outra página do aplicativo desse banco, colocando em prática meus conhecimentos em HTML5, CSS3 e JavaScript (principalmente DOM).

Vídeo de demonstração

Tecnologias Usadas

  • HTML5
  • CSS3
  • JavaScript

Desafios do Projeto

1 - A ideia inicial era só criar uma página principal de um banco usando meus conhecimentos de CSS3 e HTML5 e bem pouco de JavaScript , mas conforme fui montando o site principal a vontade de criar um aplicativo simples , usando mais funcionalidades do JavaScript, foi aumentando.

2 - Por ser um banco moderno as duas páginas tinham que ser responsivas em todos os tamanhos de telas, mantendo a qualidade e funcionalidades.

3 - O aplicativo, por ser um projeto mais interativo, precisava de um feedback maior para cada ação que o usuário faria.

Minhas Soluções

1 - Decidi criar uma outra página para o aplicativo e integrar com a do site do banco, focando mais em JavaScript no app.

2 - Aproveitei para aprender mais sobre CSS3 e além de aprender a criar páginas mais responsivas, comecei a entender mais sobre flexbox e grid e apliquei tudo nos dois projetos.

3 - Criei uma janela de feedback para alertar o usuário se algo foi realizado com sucesso ou não, dando soluções para as ações que deram errado.

Funcionalidades Aplicativo

  • Histórico de movimentação do dinheiro
  • Transferir dinheiro entre as contas
  • Pedir "Empréstimo"
  • Login dos Usuários
  • Limite de tempo para cada seção
  • Janela de Feedback
  • "Deletar" conta
