ASA Ação Solidária Adventista

Projeto Assistência Social

Resumo

Projeto que eu fiz para o departamento de Assistência Social da igreja que frequento, Igreja Adventista do Sétimo Dia – Central Indaiatuba , para divulgar as ações sociais em andamento, explicar mais sobre a instituição e engajar quem quiser contribuir ou entrar em contato com eles.

Video de demonstração

Tecnologias Usadas

  • HTML5
  • CSS3
  • JavaScript
  • React JS
  • Next.js
  • Tailwind CSS

Desafios do Projeto

Esse projeto testou meu conhecimento para arquitetar um site para um cliente pela primeira vez, com prazo, necessidades, exigências e requisitos reais.

Minhas Soluções

Apliquei meus conhecimentos de Engenharia de Software para levantar os Requisitos Funcionais, Não Funcionais e necessidades, através da entrevista com o cliente. E a cada iteração, mostrava o progresso do projeto a ele, recebia o feedback e fazia as alterações necessárias para progredir.
