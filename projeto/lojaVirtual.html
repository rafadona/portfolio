Projeto Loja Virtual

Ecommerce

Resumo

Meu primeiro projeto Full Stack, simulando uma Loja Virtual, para colocar em prática meus conhecimentos em tecnologias mais novas.

Video de demonstração

Tecnologias Usadas

  • React
  • Firebase
  • Next.js
  • Redux
  • Tailwind CSS
  • RESTful API
  • Autenticação OAuth 2.0
  • Vercel para hospedar

Desafios do Projeto

1 - Alimentar o site com produtos.

2 - Criar componentes reutilizáveis para cada parte do site.

3 - Usar CSS3 puro seria muito complicado para estilizar todo o site, eu precisava de alguma ferramenta para facilitar todo esse trabalho.

4 - Gerar dinamicamente uma página para cada produto.

5 - Implementar algum tipo de autenticação para simular um cenário real.

6 - Armazenar as informações do pedido finalizado em algum lugar.

7 - Usar as informações armazenadas no Firebase para criar uma pagina de histórico de pedidos finalizados.

8 - Eu precisava de alguma ferramenta que guardasse os itens no "carrinho" de forma global, podendo acessar essas informações em qualquer componente que eu quisesse.

Minhas Soluções

1 - Decidi usar a Fake Store API, uma API em inglês, para ser a minha fonte de dados dos produtos.

2 - Aprendi e coloquei em prática meus conhecimentos em React e modulazirei todo o site em pequenos componentes.

3 - Conheci e aprendi a usar a ferramenta Tailwind CSS, que facilitou muito todo o trabalho estético do site.

4 - Aprendi a usar o Next.js primeiramente só para usar a função de Router e criação de páginas dinâmicas, no meio do caminho aprendi que o Next.js tinha funções de otimização de imagens e também acabei usando esse recurso.

5 - Implementei a solução de autenticação do Google, Autenticação OAuth 2.0.

6 - Aprendi a implementar o Firestore Database para armazenar as informações de cada pedido finalizado.

7 - Aprendi a buscar as informações de cada pedido finalizado no Firebase, e criei uma página "Meus Pedidos".

8 - Aprendi a usar a ferramenta Redux para gerenciar o estado do aplicativo e armazenar globalmente as informações do "carrinho".

Funcionalidades

  • Autenticação pelo Google
  • Feed de produtos através de uma API
  • Página com detalhes para cada produto
  • Carrinho global
  • Página de Checkout
  • Página "Meus Pedidos"
  • Armazenamento das informações em banco de dados
  • Site Responsivo
